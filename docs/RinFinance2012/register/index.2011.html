Thursday Workshop Details

This full-day workshop is offered on Thursday preceding the actual conference, and will run from 09:00am to 05:00pm. The cost for this optional workshop is $500.

  1. R / C++ integration with Rcpp and RInside
    Dirk Eddelbuettel and Romain Francois will provide a full-day workshop on integrating R and C++ using the Rcpp and RInside packages.

Pre-Conference Tutorial Details

There are three two-hour tutorials offered on Friday morning from 09:00 to 11:00am. The cost is $75, and the tutorials are offered in parallel so only one can be selected per participant.

  1. Algorithmic Trading with R
    Jeff Ryan will explore using R within the context of a rapid prototyping and trade execution system.
  2. High-Frequency Financial Data Analysis with R
    Guy Yollin and Scott Payseur offer a hands-on introduction to using R for high-frequency financial data analysis.
  3. Financial Risk Models with R: Factor Models for Asset Returns and Interest Rate Models
    Eric Zivot will focus on two core areas of financial analysis and provide examples in the statistical programming language R.

Simply register for the workshop and/or one of the tutorials, or select none, on the secure registration page. Pre-conference tutorials and workshops are optional.
